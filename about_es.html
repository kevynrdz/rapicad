RAPICAD es una base de datos con modelos CAD paramétricos de objetos, elementos de sujeción, herramientas y un sin fin de productos disponibles para su uso en proyectos de diseño.


Todos los modelos están hechos con FreeCAD, el cual es un software libre.

Cuando comencé a diseñar, no había muchas opciones conocidas de software de código abierto para CAD. Hasta ahora no existía una biblioteca abierta de modelos 3D de tornillería y componentes estándar realizados con software libre.

RAPICAD busca agilizar el proceso de diseño brindando modelos CAD paramétricos estandarizados para que tanto profesionales como principiantes puedan desarrollar sus proyectos de una manera más sencilla.

Finalmente, gracias a todos por apoyar este proyecto.
